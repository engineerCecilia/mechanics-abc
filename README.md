## 📝 如何补充剩余文章

目前每个分类只有部分示例文章，你可以按照以下模板补充剩余文章：

### 文章模板

复制任意一篇已有的文章（如 `categories/category01/article01.html`），然后修改以下内容：

1. **标题**：修改 `<title>` 和 `<h1>` 标签
2. **内容**：在 `<div class="article-content">` 中编写文章内容
3. **导航链接**：更新上一篇/下一篇的链接

### 内容结构建议

```html
<div class="article-content">
    <h1>文章标题</h1>
    
    <div class="highlight-box">
        <strong>一句话总结：</strong>核心要点
    </div>

    <h2>第一个小节标题</h2>
    <p>段落内容...</p>
    
    <h2>第二个小节标题</h2>
    <ul>
        <li>列表项1</li>
        <li>列表项2</li>
    </ul>

    <div class="tip-box">
        <strong>💡 小贴士：</strong>提示信息
    </div>

    <div class="article-nav">
        <a href="articleXX.html">← 上一篇</a>
        <a href="articleXX.html">下一篇 →</a>
    </div>
</div>
```

## 💡 写作建议

### 文章风格
1. **通俗易懂**：避免专业术语，用生活化语言
2. **配合例子**：多用生活中的实例说明
3. **图文并茂**：可以添加图片帮助理解（目前为纯文字）
4. **重点突出**：使用高亮框强调核心概念

### 内容方向
- 基础概念类：解释"是什么"、"为什么"
